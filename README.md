# Module Gestion Entreprises - Frontend

Module frontend pour la gestion des entreprises et exercices comptables.

## Fonctionnalités

- Context React pour la gestion de l'entreprise active
- Page de sélection d'entreprise
- Gestion des exercices comptables
- Multi-entreprise

## Structure

```
src/
├── contexts/          # React contexts
│   └── EntrepriseContext.tsx
├── pages/             # Pages React
│   └── selection-entreprise.tsx
├── components/        # Composants réutilisables
└── types/            # Types TypeScript
```

## Utilisation

Ce module est conçu pour être intégré dans l'application principale Logiciel Compta.

## Technologies

- React
- TypeScript
- Context API
